## 动态内存分配new/malloc

分配的内存是在堆区**z**

### new

`new` 是 C++ 中的关键字，用于动态分配内存，并自动调用对象的构造函数

```
语法
类型* 指针 = new 类型;
类型* 指针 = new 类型(初始值); // 带初始化
类型* 指针 = new 类型[大小];  // 分配数组

例:
// 分配单个整数
    int* p = new int(10); // 分配一个整数并初始化为 10

// 分配数组
    int* arr = new int[5]; // 分配一个大小为 5 的整数数组
```

内存使用完后,要使用delete函数将已分配的内存清空

### malloc

`malloc` 是 C 语言中的函数，用于动态分配内存，但不会调用构造函数(c++中也是可以用的)

```
语法
void* 指针 = malloc(大小); // 返回 void*，需要强制类型转换

例:
// 分配单个整数
    int* p = (int*)malloc(sizeof(int)); // 分配一个整数

 // 分配数组
    int* arr = (int*)malloc(5 * sizeof(int)); // 分配一个大小为 5 的整数数组
```

内存使用完后,要使用free函数将已分配的内存清空

## 三元运算符(a > b) ? a : b

```
条件 ? 表达式1 : 表达式2;
```

* **条件** ：一个布尔表达式，返回 `true` 或 `false`。
* **表达式1** ：当条件为 `true` 时执行的代码或返回的值。
* **表达式2** ：当条件为 `false` 时执行的代码或返回的值。
